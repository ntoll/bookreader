Using this Application

Well done, you've found the help page. This proves that you're at least capable of navigating a UI and clicking on a "help" link. ;-)

As the about page states, this is a simple web-application that allows you to contribute annotations to Becky Hogge's book Barefoot into Cyberspace. Furthermore, you get to see what others have annotated (each block of text has an associated timeline of comments). As a result, this application turns the book Barefoot into Cyberspace into an evolving collaboration.

To read a chapter click the "Contents" link at the top of this page and select the chapter you wish to read. When reading a chapter click on the tag icons next to each block of text to see the annotations people have left. Click the "close" link to dismiss the annotations and return to reading the chapter. You'll sometimes see links within the text. Often these work just like regular links by taking you to another page on the web. However, some links are really references. When you click these a pop-over will appear with the relevent reference displayed inside. Try it now by clicking this link (clicking the link again hides the popup).

What you may not know is that this application is also a programming example in a book about the openly writeable data store Fluidinfo. The book's title is Getting Started with Fluidinfo and it's published by O'Reilly.

In order to contribute annotations you need to have an account on Fluidinfo. Sign up at Fluidinfo's account creation page, return here and then sign in with your username and password by clicking the login link at the top right hand side of this page.

Once logged in you'll notice that when you display the annotations there is also an Annotate this! button to display a form. Click the "Submit" button to add an annotation. When any of your annotations are displayed they will have a delete link next to them. Clicking it obviously removes the annotation. You can annotate as many times as you like on any number of blocks of text.

You can submit feedback about this application (not the work Barefoot into Cyberspace itself) at any time by clicking on the "feedback" link on the left hand side of the screen.

That's it and have fun!
